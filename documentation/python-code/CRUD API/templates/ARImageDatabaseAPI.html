Welcome to the AR Platform API

This is a guide designed to help you create content for our AR platform. These also include examples using our api.

Base Object Attributes

These attributes will appear on every object in this API. They're the most important because they tell our app how
the rest of the content will be shown. If your content is not showing, most likely there's a simple mistake in this portion
of your attribute code. These attributes need to come first before writing any other object.

"ui" : "2D",
"type" : "list",
"x-pos" : -0.2,
"y-pos" : 0.026,
"z-pos" : 0.167,

List Object 2D

type : "list"

The list object is used when you wanna display a list of items for the user to take action on.
The most one singular list can hold is three at a time. You can however use more than list on a target.
Each item in the list object contains a couple of different sub elements: Title, subtitle, detail, button.
There are multiple attributes that you can control on each element, but for the time being it is a little simple.

Attributes:

"list-num" : 3,
"list-title" : "UPCOMING SHOWS",

Content Object 2D

type : "content"

The content object is used when you wanna display content from another service and link to the web or deep link.
The most one singular list can hold is three at a time. You can however use more than list on a target.
Each item in the list object contains a couple of different sub elements: Title, subtitle, detail, button.
There are multiple attributes that you can control on each element, but for the time being it is a little simple.

Attributes:

"content-title" : "Ogden Theatre",
"content-subtitle" : "ogdentheatre.com",
"content-detail-bold" : "7 min",
"content-detail-small" : "1.4 miles",
"content-preview" : "images/ogden_map.png",
"content-deep-link" : "https://maps.google.com"
